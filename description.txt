백준(BOJ)과 SWEA 제출용 자바 코드를 복사·붙여넣기(Ctrl+V) 하는 순간, 제출 포맷에 맞게 한 번에 자동 정리해 주는 크롬 확장 프로그램입니다.

패키지 제거, 플랫폼별 클래스명 변경 같은 귀찮은 반복 작업을 자동화하여 제출 직전의 잔실수를 없애고, 오직 문제 풀이에만 집중할 수 있게 도와줍니다.


🤔 왜 필요한가요?
알고리즘 문제를 풀고 제출할 때마다 우리는 이런 귀찮은 작업을 반복합니다.
• 로컬 IDE(IntelliJ, Eclipse 등) 실행용 package 선언문 지우기
• 플랫폼 채점 규칙에 맞춰 메인 클래스명 변경 (Main, Solution 등)
• 깜빡하고 그대로 제출했다가 발생하는 '컴파일 에러' 겪기

작은 작업이지만, 매번 수동으로 하면 시간이 들고 실수가 생깁니다.
이 익스텐션은 별도의 버튼 클릭조차 필요 없습니다.
에디터에 코드를 복사해서 붙여넣는 즉시 모든 변환이 끝납니다.


✨ 핵심 기능
• 복사 후 붙여넣는 즉시 코드를 가로채어 자동 포맷팅 후 입력합니다.
• 로컬 환경에서 작성된 패키지 Package 선언문을 깔끔하게 지워줍니다.
• 플랫폼별 클래스명 자동 변환
   ◦ BOJ: Main
   ◦ SWEA: Solution
• 내부 클래스 완벽 보호:  main(진입점) 메서드를 감싸는 진짜 껍데기 클래스만 찾아 변경합니다.
• 주석/코드 구분 처리(안전 변환)


🎯 이런 분께 추천합니다
• 백준, SWEA를 Java로 꾸준히 풀고 계신 분
• IntelliJ 등 로컬 IDE에서 알고리즘을 풀고 플랫폼에 복붙해서 제출하시는 분
• 클래스명, 패키지 제출 실수로 인한 '컴파일 에러'나 '런타임 에러'를 겪어보신 분
• 제출 직전 코드 수정하는 잠깐의 시간마저 아끼고 싶은 분


🛠 최근 안정화 업데이트
